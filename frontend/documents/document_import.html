Как работает импорт

Файл разбирается по структуре текущего плана.

Импорт создает объекты, лицензии, направления, инстанции и документы.
  1. Объекты берутся из секций Excel.
  2. Документы берутся из колонки с названием документа.
  3. Наличие, дата получения и примечания переносятся в карточку документа.
  4. Дедлайн ищется в примечаниях или по красной зоне календаря.
